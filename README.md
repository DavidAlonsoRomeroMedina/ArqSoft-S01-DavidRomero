# Catálogo de Videojuegos - MVC

Proyecto práctico de Arquitectura de Software desarrollado en .NET utilizando el patrón de diseño Modelo-Vista-Controlador (MVC). Este proyecto demuestra la separación de responsabilidades y la personalización de la capa de la Vista (View) mediante la manipulación de archivos estáticos.

Desarrollado como parte de las actividades del 3do Cuatrimestre en el Instituto Tecnológico de Software (ITS).

## 🚀 Tecnologías Usadas

* **C# / .NET** 
* **ASP.NET Core MVC** (Arquitectura)
* **HTML5, CSS3 y JavaScript** (Diseño visual interactivo y animaciones SVG personalizadas)
* **Visual Studio Community** (Entorno de desarrollo)

## ⚙️ Organización del Código

El repositorio se mantiene limpio de binarios locales y archivos autogenerados. Se ha implementado un archivo `.gitignore` estandarizado para Visual Studio, asegurando que directorios como `bin/` y `obj/` no formen parte del control de versiones, siguiendo las mejores prácticas de un equipo de desarrollo profesional.
